## Task 8 — Preprocessing Pipeline

> ⚠️ **Before starting:** Task 7 verified. Write in `src/preprocessing.py`.

Implement `preprocess_cycle(wav_path, start, end, config)` that:

**Step 1 — Load audio segment:**
```python
audio, sr = librosa.load(wav_path, sr=config["sample_rate"], mono=True)
cycle = audio[int(start * sr) : int(end * sr)]
```

**Step 2 — Cyclic padding to fixed length (NEVER zero-pad):**
```python
target_len = int(config["target_duration"] * config["sample_rate"])
if len(cycle) < target_len:
    repeats = (target_len // len(cycle)) + 1
    cycle = np.tile(cycle, repeats)[:target_len]
else:
    cycle = cycle[:target_len]
# cycle.shape must be (target_len,) = (176400,) for 8s at 22050Hz
```

**Step 3 — Log-Mel spectrogram:**
```python
mel = librosa.feature.melspectrogram(
    y=cycle,
    sr=config["sample_rate"],
    n_mels=config["n_mels"],       # 128
    n_fft=1024,
    hop_length=512,
    fmax=8000
)
log_mel = librosa.power_to_db(mel, ref=np.max)
# Shape: (128, 345) approximately — verify this matches model expectations
```

**Step 4 — Per-sample normalization:**
```python
log_mel = (log_mel - log_mel.mean()) / (log_mel.std() + 1e-8)
```

```python
from torch.utils.data import WeightedRandomSampler

def get_weighted_sampler(dataset):
    class_counts = [3642, 1864, 886, 506]  # normal, crackle, wheeze, both
    class_weights = 1.0 / torch.tensor(class_counts, dtype=torch.float)
    sample_weights = [class_weights[label] for _, label in dataset]
    return WeightedRandomSampler(sample_weights, len(sample_weights), replacement=True)
```

## Task 13 — SAM Optimizer

> ⚠️ **Before starting:** Task 12 verified. Write in `src/optimizer.py`.  
> ⚠️ **This is a complex task. Do not simplify the two-step procedure.**

Implement the full SAM optimizer class. The two-step training loop is mandatory:

```python
class SAM(torch.optim.Optimizer):
    """
    Sharpness-Aware Minimization optimizer.
    Usage:
        optimizer = SAM(model.parameters(), torch.optim.AdamW, rho=0.05, lr=1e-4)
        
        # In training loop — TWO forward passes per batch:
        loss = criterion(model(inputs), labels)
        loss.backward()
        optimizer.first_step(zero_grad=True)

        criterion(model(inputs), labels).backward()
        optimizer.second_step(zero_grad=True)
    """
```

> ⚠️ **If your model uses BatchNorm or Dropout:**  
> Before the first forward pass, call `model.eval()` temporarily, then `model.train()` again before the second pass. This prevents BatchNorm stats from being updated twice.

**Unit test:**

```python
y = torch.randint(0, 4, (8,))

params_before = net.weight.data.clone()

# SAM two-step
loss = crit(net(x), y)
loss.backward()
opt.first_step(zero_grad=True)

crit(net(x), y).backward()
opt.second_step(zero_grad=True)

params_after = net.weight.data.clone()
assert not torch.allclose(params_before, params_after), "Weights did not update!"
print("✅ SAM optimizer verified.")
```

```python
class FocalLoss(nn.Module):
    def __init__(self, gamma=2.0, weight=None):
        super().__init__()
        self.gamma = gamma
        self.weight = weight

    def forward(self, logits, targets):
        ce_loss = F.cross_entropy(logits, targets, weight=self.weight, reduction="none")
        pt = torch.exp(-ce_loss)
        return ((1 - pt) ** self.gamma * ce_loss).mean()
```

## Task 15 — ICBHI Evaluation Metrics

> ⚠️ **Before starting:** Task 14 verified. Write in `src/evaluate.py`.

Implement the **official ICBHI evaluation protocol**. The challenge evaluates binary: Normal vs. Abnormal.

```python
def compute_icbhi_metrics(y_true: np.ndarray, y_pred: np.ndarray) -> dict:
    """
    Args:
        y_true: array of ground truth labels {0, 1, 2, 3}
        y_pred: array of predicted labels {0, 1, 2, 3}
    Returns:
        dict with keys: Se (sensitivity), Sp (specificity), Score, accuracy
    
    Protocol:
        Normal (0) → binary 0
        Crackle (1), Wheeze (2), Both (3) → binary 1 (Abnormal)
        Se = TP / (TP + FN)  → how well we catch abnormal sounds
        Sp = TN / (TN + FP)  → how well we avoid false alarms
        Score = (Se + Sp) / 2
    """
```

**Unit test with known values:**
```python

metrics = compute_icbhi_metrics(y_true, y_pred)
assert metrics["Se"] == 1.0, f"Perfect Se should be 1.0, got {metrics['Se']}"
assert metrics["Sp"] == 1.0, f"Perfect Sp should be 1.0, got {metrics['Sp']}"
assert metrics["Score"] == 1.0

# All predicted as normal (worst case for sensitivity)
y_pred_all_normal = np.zeros(6, dtype=int)

assert metrics2["Se"] == 0.0, "Predicting all normal → Se=0"
assert metrics2["Sp"] == 1.0, "Predicting all normal → Sp=1"

print("✅ ICBHI metrics verified.")
```

```python
def save_checkpoint(state, path):
    """state = {epoch, model_state_dict, optimizer_state_dict, best_Se, metrics_history}"""
    torch.save(state, path)

def load_checkpoint(path, model, optimizer):
    """Returns: start_epoch, best_Se"""
    checkpoint = torch.load(path, map_location="cpu")
    model.load_state_dict(checkpoint["model_state_dict"])
    optimizer.load_state_dict(checkpoint["optimizer_state_dict"])
    return checkpoint["epoch"] + 1, checkpoint["best_Se"]
```
